Class: Viewport

Viewport

Represents a combination of canvas and Three.js scene, camera and renderer objects.

Constructor

new Viewport(canvas, color, opacity)

Creates a viewport object in a given canvas. Functions defined in the constructor have local/private variable access.
Parameters:
Name Type Default Description
canvas HTMLElement Canvas DOM element to attach to
